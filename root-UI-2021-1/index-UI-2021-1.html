CARDS

  • I designed and developed this project with the goal of using the Card UI design along with some of the other 2020-2021 UI design trends such as: dark mode, thumb-friendly mobile navigation, smart content load through lazy loading, micro interactions, minimalism (Flat Design), and white space
  • This Card UI design uses the back of the card and almost doubles the amount of information provided per website space
  • I used RWD, HTML5, CSS3, Flexbox, and Vanilla JavaScript
  • Additional projects on GitHub; read about me, and contact me with any questions or comments
